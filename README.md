# ☁️ WuTang Dev Cloud Lab

## 📌 Overview
This repository documents my hands-on journey into Microsoft cloud technologies, building on my foundation in on-prem infrastructure.

The goal is to develop practical skills in cloud-based environments, focusing on identity, device management, and modern infrastructure using Microsoft 365, Entra ID, and Intune.

All labs are completed in structured daily sessions to maintain consistency and reinforce real-world workflows.

---

## 🧱 Lab Structure

Each lab is organised by day:

```
Day-01-Cloud-Orientation/
Day-02-Identity-Users/
Day-03-Groups-and-Access/
...
```

Each folder contains a detailed README covering:
- Objectives
- Configuration steps
- Commands used
- Key learning points

---

## 🖥️ Technologies Covered

- Microsoft 365
- Microsoft Entra ID (Azure AD)
- Microsoft Intune
- Identity & Access Management (IAM)
- Device Management
- Cloud-based infrastructure concepts

---

## 🎯 Objectives

- Transition from on-prem to cloud environments
- Build strong understanding of identity and access control
- Gain hands-on experience with Microsoft cloud tools
- Develop consistent documentation habits
- Create a portfolio of real, practical labs

---

## 🧠 Approach

This lab follows a structured approach:

- 20-minute focused sessions
- Daily GitHub updates
- Clean, readable documentation (no screenshots)
- Emphasis on understanding over memorisation
